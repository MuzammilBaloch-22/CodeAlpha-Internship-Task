# The Royal Spoon - Landing Page & AI Chatbot

Welcome to **The Royal Spoon**, a premium restaurant landing page with a **fully integrated AI-powered chatbot**. This project demonstrates a complete **full-stack MERN solution** with advanced features for user interaction, event booking, and automated communications.

---

## Project Overview

This project is composed of two main modules:

### 1. Landing Page
- Fully responsive **MERN stack** website built with **React + TypeScript + HTML/CSS/JS**.
- Multi-page architecture:
  - **Home**
  - **About**
  - **Menu**
  - **Testimonials**
  - **Chef**
  - **Contact**
- Interactive forms to capture user data and deliver it directly to the admin email.
- Smooth scrolling, transitions, and animations for an enhanced user experience.

### 2. AI Chatbot
- Handles **event booking inquiries** directly on the website.
- Dynamically collects user information through conversation flow.
- Sends **confirmation emails** to users using **NodeMailer**.
- Powered by **Dialogflow knowledge base** with FAQs imported via CSV.
- Fully integrated with a backend API (`Node.js` + `Express.js`), deployed as `Dialogflow.server` on Render.

---

## Key Features

- **Event Booking Automation**
  - Users can click "Book for Event" to interact with the chatbot.
  - Guided dialogue collects relevant booking information.
  - Sends confirmation emails automatically.

- **Knowledge Base Integration**
  - Dialogflow processes FAQs from CSV files for fast, automated responses.

- **Form Submission & Data Handling**
  - Forms on all pages submit user data securely via backend API.
  - Admin receives email notifications for each submission.

- **Modern Frontend Experience**
  - React TSX components with responsive layouts.
  - Smooth scrolling and animations powered by Framer Motion.
  - Clean, professional UI/UX design.

- **Robust Backend**
  - Node.js + Express API to manage chatbot requests and form submissions.
  - NodeMailer integration for reliable email delivery.
  - Dialogflow server deployment on Render ensures scalable chatbot services.

---


## Tech Stack

- **Frontend:** React, TypeScript, HTML5, CSS3, JavaScript, Framer Motion  
- **Backend:** Node.js, Express.js, Dialogflow, NodeMailer  
- **Database:** Not required (data sent via email)  
- **Deployment:** Render (Dialogflow server)

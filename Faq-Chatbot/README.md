## üìñ Project Overview

**The Royal Spoon** is a high-end restaurant landing page integrated with a custom-built **AI Chatbot**. It bridges the gap between premium web design and automated customer service, handling everything from menu exploration to real-time event bookings.

---

## üõ†Ô∏è Key Modules & Features

### 1. üé® The Landing Page (MERN Frontend)
A fully responsive, high-performance website built with **React and TypeScript**.
* **Dynamic Sections:** Home, About, Menu, Testimonials, Chef, and Contact.
* **User Experience:** Smooth scrolling and premium animations using **Framer Motion**.
* **Form Integration:** Captures user queries and delivers them directly to the admin via a secure backend.

### 2. ü§ñ AI Chatbot (Dialogflow Engine)
The heart of the project‚Äîan intelligent agent that automates customer engagement.
* **Event Booking:** Guided conversation flow to collect guest details, dates, and event types.
* **Knowledge Base:** Powered by Dialogflow, utilizing CSV-imported data to answer FAQs instantly.
* **Automated Communication:** Integrated with **NodeMailer** to send instant booking confirmations to users.
* **Backend Support:** Deployed on **Render** using Node.js and Express.js.

## ‚öôÔ∏è Installation & Setup

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/MuzammilBaloch-22/The-Royal-Spoon.git](https://github.com/MuzammilBaloch-22/The-Royal-Spoon.git)
    ```
2.  **Install Frontend Dependencies:**
    ```bash
    cd frontend && npm install
    ```
3.  **Install Backend Dependencies:**
    ```bash
    cd backend && npm install
    ```
4.  **Configure Environment Variables:** Create a `.env` file in the backend for `DIALGOFLOW_KEYS` and `NODEMAILER_AUTH`.

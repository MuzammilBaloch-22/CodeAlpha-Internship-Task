## üìÇ Project Showcase

Here are the three core tasks completed during this internship, showcasing a blend of AI backends and modern web frontends.

### 1. üéµ AI Music Generator
An intelligent application that composes melodies using mathematical logic and music theory libraries.
* **Backend:** Python (Flask), `music21`, `numpy`
* **Frontend:** HTML, CSS, Tailwind CSS
* **Features:** Generates musical notes and MIDI files programmatically, served via a clean Tailwind-styled interface.

![Flask](https://img.shields.io/badge/Flask-000000?style=flat-square&logo=flask&logoColor=white)
![NumPy](https://img.shields.io/badge/numpy-%23013243.svg?style=flat-square&logo=numpy&logoColor=white)
![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=flat-square&logo=tailwind-css&logoColor=white)

### 2. ü§ñ FAQ Chatbot
A smart conversational agent built to handle user queries efficiently using Google's Dialogflow for Natural Language Understanding.
* **Core Logic:** Google Dialogflow
* **Frontend:** React.js, TypeScript
* **Features:** Provides instant responses to Frequently Asked Questions with a modern, type-safe user interface.

![Dialogflow](https://img.shields.io/badge/Dialogflow-FF9800?style=flat-square&logo=dialogflow&logoColor=white)
![React](https://img.shields.io/badge/React-20232A?style=flat-square&logo=react&logoColor=61DAFB)
![TypeScript](https://img.shields.io/badge/TypeScript-007ACC?style=flat-square&logo=typescript&logoColor=white)

### 3. üåê Language Translator
A powerful translation tool leveraging the capabilities of Large Language Models (LLMs) for high-accuracy text conversion.
* **AI Engine:** Google Gemini API
* **Language:** TypeScript
* **Features:** Real-time translation between multiple languages utilizing Generative AI for nuanced understanding.
